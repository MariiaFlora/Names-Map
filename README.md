# Names-Map

## Опис проекту
**Names-Map** — це Java-додаток, який демонструє використання інтерфейсу `Map` для роботи з даними про користувачів. Додаток надає можливість отримати список всіх користувачів або інформацію про конкретного користувача за його ідентифікатором.

### Основні класи
1. **Main.java** — основний клас програми, який викликає методи для отримання даних і передає їх для виводу.
2. **UIOperator.java** — клас, відповідальний за виведення інформації в консоль.
3. **DataRepository.java** — клас, що зберігає дані про користувачів у вигляді об’єкта `Map`.
4. **DataHandler.java** — клас, що обробляє дані з `DataRepository` і форматує їх для виводу.

### Як працює програма
1. Клас `DataRepository` створює і повертає об’єкт `Map`, що містить ідентифікатори та імена користувачів.
2. Клас `DataHandler` надає методи для:
   - Отримання списку всіх користувачів у нумерованому форматі.
   - Отримання інформації про конкретного користувача за його ідентифікатором.
3. Основний клас `Main` ініціалізує обробник даних (`DataHandler`) і передає результати до `UIOperator` для виводу.

### Вимоги до системи
- Java 8 або вище.
- Середовище розробки для Java (VSCode, IntelliJ IDEA, Eclipse).
